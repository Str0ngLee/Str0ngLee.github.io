дҪ пјҲжөҸи§ҲеҷЁпјүдёҚзӣҙжҺҘи®ҝй—®еҗҺз«ҜжңҚеҠЎеҷЁпјҢиҖҢжҳҜе…Ҳи®ҝй—® NginxпјҲе®Ҳй—ЁдәәпјүпјҢNginx еҶҚеё®дҪ еҺ»еҗҺз«ҜжӢҝж•°жҚ®еӣһжқҘз»ҷдҪ гҖӮ

еҘҪеӨ„пјҡ

  • йҡҗи—ҸеҗҺз«ҜжңҚеҠЎеҷЁпјҢеҸӘжҡҙйңІ NginxпјҢжӣҙе®үе…Ё

  • з»ҹдёҖе…ҘеҸЈпјҢж–№дҫҝеҒҡиҙҹиҪҪеқҮиЎЎ

  • SSL иҜҒд№ҰгҖҒзј“еӯҳгҖҒgzip еҺӢзј©зӯүеҸҜд»Ҙз»ҹдёҖеңЁ Nginx еұӮеӨ„зҗҶпјҢеҗҺз«ҜдёҚз”Ёз®Ў

иҪ®иҜўпјҲRound-RobinпјүжҳҜд»Җд№Ҳпјҹ

е°ұеғҸеҸ‘жү‘е…ӢзүҢдёҖж ·пјҢжҜҸдәәдёҖеј иҪ®жөҒжқҘпјҡ

Nginx й»ҳи®Өе°ұжҳҜиҝҷз§ҚеҲҶй…Қж–№ејҸпјҢж— йңҖйўқеӨ–й…ҚзҪ®пјҢеұһдәҺејҖз®ұеҚіз”ЁгҖӮ

рҹҸ—пёҸ жһ¶жһ„жҖ»и§Ҳ

дёҖеҸ° Nginx жҢЎеңЁеүҚйқўпјҢеҗҺйқўжҢӮ N еҸ°е®Ңе…ЁдёҖж ·зҡ„еҗҺз«ҜжңҚеҠЎгҖӮз”ЁжҲ·еҸӘзңӢеҲ° NginxпјҢдёҚзҹҘйҒ“еҗҺйқўжңүеҮ еҸ°жңәеҷЁгҖӮ

рҹӘң ж“ҚдҪңжӯҘйӘӨ

е…Ҳз”ЁдҪ жңҖзҶҹжӮүзҡ„иҜӯиЁҖеҗҜеҠЁеҮ дёӘеҗҺз«ҜгҖӮд»Ҙ Node.js дёәдҫӢпјҢеҲӣе»әеӣӣдёӘзӣ®еҪ•пјҢеҗ„ж”ҫдёҖдёӘжңҖз®ҖеҚ•зҡ„жңҚеҠЎеҷЁпјҡ

жҜҸдёӘзӣ®еҪ•дёӢж”ҫдёҖдёӘ server.jsпјҢе”ҜдёҖзҡ„еҢәеҲ«жҳҜз«ҜеҸЈе’Ңиҝ”еӣһзҡ„ж ҮиҜҶж–Үеӯ—пјҡ

еҸҰеӨ– node2/server.jsгҖҒnode3/server.jsгҖҒnode4/server.js еҗҢзҗҶпјҢеҸӘйңҖиҰҒж”№з«ҜеҸЈеҸ·пјҲ3002/3003/3004пјүе’ҢжҳҫзӨәж–Үеӯ—пјҲиҠӮзӮ№ B/C/DпјүеҚіеҸҜгҖӮ

Python е°ұжӣҙз®ҖеҚ•дәҶпјҢдёҖиЎҢжҗһе®ҡпјҲеҗҺйқўдјҡи®ІпјүгҖӮ

жӯҘйӘӨ 2пёҸвғЈпјҡеҗҜеҠЁеҗҺз«ҜжңҚеҠЎ

ејҖеӣӣдёӘз»Ҳз«ҜзӘ—еҸЈпјҢеҲҶеҲ«иҝҗиЎҢпјҡ

йӘҢиҜҒжҳҜеҗҰеҗҜеҠЁжҲҗеҠҹпјҡ

жөҸи§ҲеҷЁи®ҝй—® http://localhost:3001 иғҪзңӢеҲ°йЎөйқўе°ұиҜҙжҳҺеҚ•еҸ°еҗҺз«ҜжӯЈеёёгҖӮ

рҹ’Ў еҰӮжһңдёҚжғіеҶҷ Node.jsпјҢз”Ё Python иҮӘеёҰзҡ„з®Җжҳ“жңҚеҠЎеҷЁдёҖиЎҢе°ұиЎҢпјҡ

python -m http.server дёҚйңҖиҰҒиЈ…д»»дҪ•дёңиҘҝпјҢжҠҠеҪ“еүҚзӣ®еҪ•еҪ“зҪ‘з«ҷж №зӣ®еҪ•пјҢи®ҝй—® / иҝ”еӣһ index.htmlгҖӮ

жӯҘйӘӨ 3пёҸвғЈпјҡзј–еҶҷ Nginx й…ҚзҪ®пјҲж ёеҝғпјҒпјү

Linux дёӢй…ҚзҪ®ж–Үд»¶йҖҡеёёеңЁ /etc/nginx/nginx.conf жҲ– /etc/nginx/conf.d/ дёӢгҖӮWindows еҲҷеңЁ Nginx е®үиЈ…зӣ®еҪ•зҡ„ conf/ дёӢгҖӮ

вҡ пёҸ е…ій”®пјҡжј”зӨәиҪ®иҜўж—¶еҝ…йЎ»еҠ  Cache-Control: no-storeпјҢеҗҰеҲҷжөҸи§ҲеҷЁдјҡзј“еӯҳиҝ”еӣһеҶ…е®№пјҢдҪ еҲ·ж–°ж°ёиҝңзңӢеҲ°еҗҢдёҖеҸ°жңҚеҠЎеҷЁгҖӮ

рҹҹў A вҶ’ рҹ”ө B вҶ’ рҹҹ  C вҶ’ рҹҹЈ D вҶ’ рҹҹў еҫӘзҺҜвҖҰ

рҹ’Ў Nginx жңҖеӨ§зҡ„дјҳеҠҝпјҡдҝ®ж”№й…ҚзҪ®еҸӘйңҖ reloadпјҢдёҚдёӯж–ӯжңҚеҠЎпјҢйӣ¶еҒңжңәжӣҙж–°гҖӮ

рҹ§Ә иҝӣйҳ¶пјҡеӣӣз§ҚиҙҹиҪҪеқҮиЎЎзӯ–з•Ҙ

еңЁ upstream еқ—йҮҢеҠ дёҖиЎҢе°ұиғҪеҲҮжҚўзӯ–з•Ҙпјҡ

| зӯ–з•Ҙ | е…ій”®еӯ— | йҖӮеҗҲеңәжҷҜ |

|вҖ”вҖ”|вҖ”вҖ”вҖ“|вҖ”вҖ”вҖ”-|

| иҪ®иҜў | й»ҳи®Ө | йҖҡз”Ёж— зҠ¶жҖҒжңҚеҠЎпјҢжңҚеҠЎеҷЁй…ҚзҪ®дёҖж · |

| еҠ жқғиҪ®иҜў | ж”№ weight | жңҚеҠЎеҷЁй…ҚзҪ®дёҚеҗҢпјҲеҘҪзҡ„еӨҡжүӣзӮ№пјү |

| жңҖе°‘иҝһжҺҘ | least_conn; | й•ҝиҝһжҺҘгҖҒWebSocketгҖҒж–Үд»¶дёҠдј  |

| IP е“ҲеёҢ | ip_hash; | йңҖиҰҒ SessionпјҢз”ЁжҲ·дёҚиғҪи·іеҲ°еҲ«зҡ„жңәеҷЁ |

рҹҸЁ е»¶дјёпјҡNginx иҝҳиғҪеҒҡд»Җд№Ҳ

жң¬ж–Үи®Ізҡ„жҳҜеҸҚеҗ‘д»ЈзҗҶ + иҙҹиҪҪеқҮиЎЎпјҢдҪҶ Nginx зҡ„иғҪеҠӣиҝңдёҚжӯўиҝҷдәӣпјҡ

иҷҡжӢҹдё»жңә вҖ” дёҖдёӘз«ҜеҸЈи·‘еӨҡдёӘзҪ‘з«ҷ

и·Ҝеҫ„еҲҶжөҒ вҖ” еҗҢдёҖеҹҹеҗҚдёӢдёҚеҗҢжЁЎеқ—

SSL/HTTPS з»Ҳз»“

еҸҚеҗ‘д»ЈзҗҶе…ёеһӢеңәжҷҜжұҮжҖ»

рҹҺҜ жҖ»з»“

жң¬ж–ҮиҰҶзӣ–дәҶ Nginx жңҖе®һз”Ёзҡ„дёүдёӘеҠҹиғҪпјҡ

  1. еҸҚеҗ‘д»ЈзҗҶ вҖ” дҪ дёҚйңҖиҰҒзҹҘйҒ“еҗҺз«ҜеңЁе“ӘпјҢNginx её®дҪ еҺ»жӢҝ

  2. иҙҹиҪҪеқҮиЎЎ вҖ” еӨҡеҸ°еҗҺз«ҜиҪ®жөҒжүӣпјҢй…ҚзҪ®дёҚиЎҢиҝҳиғҪеҠ жқғ

  3. зғӯйҮҚиҪҪ вҖ” ж”№й…ҚзҪ®ж— йңҖеҒңжңәпјҢз”ҹдә§зҺҜеўғеҸӢеҘҪ

еӯҰе®ҢиҝҷдәӣпјҢдёҖеҸ°жңҚеҠЎеҷЁдёҠи·‘еӨҡдёӘйЎ№зӣ®гҖҒжЁӘеҗ‘жү©е®№йғҪдёҚеңЁиҜқдёӢгҖӮдёӢдёҖжӯҘеҸҜд»Ҙж·ұе…Ҙзҡ„ж–№еҗ‘пјҡ

  • proxy_cache вҖ” еҸҚеҗ‘д»ЈзҗҶзј“еӯҳпјҢеҮҸиҪ»еҗҺз«ҜеҺӢеҠӣ
  • proxy_buffer вҖ” зј“еҶІи°ғдјҳпјҢеә”еҜ№еӨ§ж–Үд»¶еңәжҷҜ
  • health_check вҖ” иҮӘеҠЁжЈҖжөӢеҗҺз«ҜеҒҘеә·зҠ¶жҖҒпјҢжҢӮдәҶиҮӘеҠЁиёўжҺү
  • Nginx + Docker Compose вҖ” е®№еҷЁзј–жҺ’пјҢдёҖй”®жӢүиө·ж•ҙеј жӢ“жү‘
  • limit_req / limit_conn вҖ” йҷҗжөҒйҳІеҲ·
